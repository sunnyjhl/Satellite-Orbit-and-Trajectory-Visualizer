# 🛰 Satellite Orbit and Trajectory Visualizer

## 📌 Overview

Satellite Orbit and Trajectory Visualizer is an interactive web-based application that simulates and visualizes satellite movement around Earth.
The project allows users to observe orbital paths, analyze satellite trajectories, and explore different types of Earth orbits in a dynamic 3D environment.

This project is designed for students, space enthusiasts, and researchers interested in orbital mechanics and satellite motion.

---

## 🚀 Features

### 🌍 3D Earth Visualization

* Interactive 3D model of Earth
* Satellites orbiting around the planet in real time

### 🛰 Satellite Orbit Simulation

Visualize different satellite orbit types:

* Low Earth Orbit (LEO)
* Medium Earth Orbit (MEO)
* Geostationary Orbit (GEO)
* Polar Orbit

### 📡 Real-Time Satellite Data

* Fetch satellite orbital data
* Display satellite position and movement

### 📊 Orbital Parameters

Display important satellite parameters:

* Altitude
* Velocity
* Inclination
* Orbital Period

### 📈 Trajectory Simulation

Users can simulate trajectories by adjusting:

* Launch angle
* Velocity
* Altitude

## ⚙ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/satellite-orbit-visualizer.git
```

### 2️⃣ Navigate to project directory

```
cd satellite-orbit-visualizer
```

### 3️⃣ Install dependencies

Frontend

```
cd frontend
npm install
```

Backend

```
cd backend
npm install
```

---

## ▶ Running the Project

Start Backend Server

```
cd backend
npm start
```

Start Frontend

```
cd frontend
npm start
```

Application will run on:

```
http://localhost:3000
```
